# 简介

WinToUSB 是一个免费的应用软件。 它可以将 Windows 操作系统从 ISO 光盘镜像或者 CD/DVD 光驱安装到 USB 移动硬盘或者 U 盘，因此您可以在 USB 移动硬盘或者 U 盘上运行 Windows 操作系统。
