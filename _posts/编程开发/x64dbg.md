# 简介

x64dbg 是一款专业的 windows 系统下的 64 位调试器，界面简洁、操作简单，与“OllyDbg”调试工具非常相似，如果之前使用过 OllyDbg 这款调试工具的朋友，可以直接上手使用。 ... GUI 是调试器的图形部分，建立在 Qt 之上，主要是提供用户交互。
