# 使用说明

1. 运行卡巴斯基，依次点击 `设置 -> 常规 -> 自我保护 -> 取消勾选启用自我保护 -> 保存`
2. 完全退出卡巴斯基
3. 运行 Kaspersky Tweak Assistant
4. 点击 Reset
5. 等待电脑重启
